# Flames
It is a webpage build with PHP, HTML. It is based on the popular school game 'FLAMES'.
You enter your name and your crush's name and it tells you your relationship status with them -
 Friend,
 Lover,
 Attraction,
 Marriage,
 Enemy or
 Sibling
